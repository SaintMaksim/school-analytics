

## О проекте

Проект разработан в рамках учебного задания по курсу "Программирование на Java". Приложение анализирует данные об образовательных учреждениях Калифорнии, выполняет аналитические запросы и визуализирует результаты.

### Анализируемые данные
- Информация о школах (название, округ, количество студентов)
- Финансовые показатели (расходы, доходы)
- Академические показатели (математика, чтение, английский)

##  Функции проекта

| Функция | Описание |
|---------|----------|
| **Загрузка данных** | Импорт из CSV файла `schools.csv` |
|  **Работа с БД** | Сохранение данных в SQLite |
|  **Аналитика** | Выполнение 3 аналитических запросов |
|  **Визуализация** | Построение диаграмм с JFreeChart |
|  **GUI** | Удобный интерфейс на Swing |
|  **Тестирование** | Полное покрытие unit-тестами |
|  **Архитектура** | Чистая архитектура MVP |

##  Архитектура MVP

- **Model** - только данные и бизнес-логика
- **View** - только отображение, без логики
- **Presenter** - связь между View и Model
- **Интерфейсы** - четкое разделение ответственности

## Установка и запуск

### Предварительные требования
- **Java JDK 17+**
- **Gradle 7.6+**

### Быстрый старт
```bash
# 1. Клонирование репозитория
git clone https://github.com/ваш-username/school-analytics.git
cd school-analytics

# 2. Сборка проекта
./gradlew build

# 3. Запуск приложения
./gradlew run
```
